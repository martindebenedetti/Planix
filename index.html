¿Qué es este planificador?

Este planificador de tareas permite organizar y visualizar el cronograma de un proyecto mediante un diagrama de Gantt. Cada tarea se representa como una barra horizontal que indica su duración, fechas de inicio y fin, y porcentaje de avance.

La herramienta está inspirada en aplicaciones como Microsoft Project y está desarrollada como proyecto académico en el marco de Programación Web I — UCES 2026.

Para comenzar, agregá una nueva tarea usando el formulario de abajo, completando el nombre, responsable, fechas y porcentaje de avance. Las tareas se verán reflejadas automáticamente en el diagrama de Gantt.

Información del Proyecto

Acerca del Proyecto
Planix es un planificador de tareas mediante un diagrama de Gantt. Permite organizar y visualizar el cronograma de un proyecto. Desarrollado por Prog. Web I - UCES 2026.
Cómo usar el Planificador

1. Agregar tarea: Usa el formulario para agregar tareas indicando nombre, fechas, responsable y porcentaje de avance.

2. Ver progreso: La tabla mostrará la barra de progreso correspondiente al porcentaje ingresado.

3. Interpretar las barras: Visualiza las fechas a través de las columnas semanales.

Leyenda del Diagrama
  • Barra azul: Indica el progreso de una tarea individual.
  • Barra gris oscura / verde: Indica un grupo de tareas.
  • Fondo celeste: Indica las columnas de las semanas.
